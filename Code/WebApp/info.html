Look at M.E. - Info

Look at M.E. is an IoT device which is able to improve the user experience for the museum visitors and to provide important data to the museum curator.

Look at M.E. use a HC-SR04 proximity sensor implemented in a Elegoo UNO R3 board capable of activating a light-guided tour supported also by an audio explaination of the art pieces in order to emphasize differents feature of an art display.
Moreover, it can build a table to show in real time the distance to the statues of the museum visitors, to make sure that no one is getting too close to an art piece.
The project's development will be reported following each step and new implementation, in fact in the GitHub repository you can find the documents of the present state of the product as well as the older markdown files and video.

The Web Application is structured to show the real time data taken from the sensors as well as a history of the measurements in the last 24 hours.




  1. Home - Real-time data table for each statue in the museum showing important data for the security of the art pieces.
  2. History - History of the measurements for each statue with number of infractions and other important informations for the last 24 hours.
  3. Info - Page containing informations about the project and web application.
  4. About - Page containing informations about the context of Look at M.E., the team and useful links.
